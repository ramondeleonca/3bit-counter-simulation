# Simple Arduino simulated 3 bit counter

A simple simulated JK flip-flop 3 bit counter with the sequence 0-2-4-6-8-10-12-14

Features:
- Buzzer feedback
- Counter display LEDs breathe animation
- Simple one-button input
- Actual JK flip-flop simulation
- Async buzzer actuation
- <15% flash
- <11% RAM
